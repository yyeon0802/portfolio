web project BinFree

지역 / 이웃 기반 대리 분리수거 서비스 구독 WEB
Idea

최근 서비스의 추세인, 지역간 소규모 중심의 matching service를 프로젝트를 통하여 이해하고자 함
- 우편번호를 활용한 matching service 구현

세계적인 문제인 코로나로 인해 변화된 삶의 모습을 가장 큰 이슈로 선정
- 문제1) 필수인 마스크와 배달식음료가 늘어감에 따라 급증하는 쓰레기
- 문제2) 대면을 최소한으로 줄여하는 일상
이러한 문제를 '편리하게' 아우를 수 있는 서비스를 만들어보고자 함.

Concept

바쁘고 지친 현대인, 사용자 BFamily
이웃이자 대리 분리수거 서비스 도우미 Buddy
서비스 구독으로 분리수거를 자주, 편리하게 !

   비밀번호 수정을 분리한 이유?
       같은 번호라도 매번 암호화를 새로하며, 빈칸도 암호화하는 경우가 발생하므로,
      사용자가 수정하고 싶을 경우에만 사용하게 하고 빈칸 입력을 방지, 가입시와 동일한 유효성 검사를 위해 분리함.

사용자가 수정전 비밀번호를 확인 할 수 있도록 현 input type인 password를 text로 변화시키는 JS
